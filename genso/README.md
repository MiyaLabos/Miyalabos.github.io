# 周期表 穴埋めプリント

ブラウザ上で動作する、元素記号学習用の印刷アプリです。  
A4縦向きで、次の2枚をまとめて出力できます。

1. 問題用紙＋回答欄
2. 模範解答

## できること

- 周期表を使った元素記号の穴埋め問題を自動生成する
- 固定された30元素からランダムで10問・20問・30問を出題できる
- 問題マスに丸付き番号を表示する
- 回答欄を別欄で印刷する
- 問題用紙と対応した模範解答を2枚目に出力する
- ブラウザの印刷機能でそのまま A4 縦向き印刷する

## 出題ルール

出題対象は全元素ではなく、次の30元素に固定しています。

- 水素 `H`
- ヘリウム `He`
- リチウム `Li`
- ベリリウム `Be`
- ホウ素 `B`
- 炭素 `C`
- 窒素 `N`
- 酸素 `O`
- フッ素 `F`
- ネオン `Ne`
- ナトリウム `Na`
- マグネシウム `Mg`
- アルミニウム `Al`
- ケイ素 `Si`
- リン `P`
- 硫黄 `S`
- 塩素 `Cl`
- アルゴン `Ar`
- カリウム `K`
- カルシウム `Ca`
- 銅 `Cu`
- 亜鉛 `Zn`
- 銀 `Ag`
- 鉄 `Fe`
- スズ `Sn`
- バリウム `Ba`
- 白金 `Pt`
- 金 `Au`
- 水銀 `Hg`
- 鉛 `Pb`

この30元素の中から毎回ランダムに 10個・20個・30個 を選び、原子番号の小さい順で出題番号を付けます。  
周期表の表示自体は全元素を維持します。

## 使い方

1. `index.html` をブラウザで開く
2. 問題数を `10問` `20問` `30問` から選ぶ
3. `新しい問題を作る` を押す
4. 内容を確認する
5. `2枚まとめて印刷` を押す
6. ブラウザの印刷設定で `A4 / 縦向き` を選んで印刷する

## 印刷仕様

- 1枚目: 問題用紙
  - 名前欄あり
  - 点数欄あり
  - 周期表の10か所・20か所・30か所が穴開き
  - 回答欄は別欄
- 2枚目: 模範解答
  - 同じ問題番号に対応する正答を表示

## ファイル構成

- `index.html`
  - 画面の土台と操作ボタン
- `styles.css`
  - 画面表示と印刷レイアウト
- `app.js`
  - 元素データ、出題生成、描画処理

## 備考

- 依存ライブラリは使っていません
- ローカルでそのまま動く静的アプリです
- 印刷結果はブラウザごとに微差が出る可能性があります

## 改善メモ

- 出題条件を画面から切り替えられるようにする
  - 例: ランタノイドを含める、典型元素だけ出す、範囲を指定する
- 問題番号の並び順を調整できるようにする
  - 現在は原子番号順で 1 から問題数までを付与している
- 回答欄のレイアウトを切り替えられるようにする
  - 例: 2列固定、4列表示、広めの記入欄
- 印刷ヘッダーを拡張する
  - 例: 日付欄、クラス欄、問題タイトルの任意編集
- 印刷前プレビューの切り替えを追加する
  - 例: 問題用紙のみ表示、模範解答のみ表示
- 出題セットを固定保存できるようにする
  - 同じ問題番号を再表示・再印刷できるようにする
- 元素データを外部ファイル化する
  - 現在は `app.js` に直接保持している
- ブラウザ差異の確認を進める
  - Chrome、Safari などで A4 縦印刷時の縮尺差を確認する
