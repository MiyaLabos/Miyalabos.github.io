# 立方体 展開図マスター (Cube Net Master)

立方体の展開図を学習するためのWebアプリケーションです。
6つの面を組み合わせて展開図を作成し、それが正しく立方体に組み上がるかをリアルタイムで検証・3D表示することができます。

## 機能
- **展開図エディタ**: 6x6のグリッド上でクリックして直感的に展開図を作成できます。
- **リアルタイム検証**:
    - 面の数が6つあるか
    - 全ての面がつながっているか
    - **【特徴】** 組み立てたときに面が重ならないか（高度な幾何計算による検証）
- **3Dビジュアライザー**:
    - 作成した展開図がどのように折り畳まれるかを3Dアニメーションで確認できます。
    - スライダー操作で任意の角度まで折りたたみ可能。
- **タブレット対応**: iPadなどのタブレット端末でも操作しやすいレイアウトとデザイン。

## 動作環境
- モダンブラウザ (Chrome, Safari, Edge, Firefox)
- ※ES Modulesを使用しているため、ローカルで動作させる場合はローカルサーバーが必要です（ファイルを直接ダブルクリックしても動作しない場合があります）。

## 実行方法
1. このディレクトリでターミナルを開きます。
2. 以下のコマンドでローカルサーバーを立ち上げます（Python 3系がインストールされている場合）。
   ```bash
   python3 -m http.server 8080
   ```
3. ブラウザで以下のURLにアクセスします。
   [http://localhost:8080](http://localhost:8080)

## 技術スタック
- **Frontend**: HTML5, CSS3 (Modern Vanilla CSS)
- **Logic**: Vanilla JavaScript (ES6+)
- **3D Library**: Three.js (via CDN)
