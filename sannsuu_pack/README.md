# 算数プリント集（sannsuu_pack）

小学生向けの算数プリントを、ブラウザだけで自動生成して印刷できる静的ページ集です。  
`index.html` を開くと、各プリントをタブで切り替えて利用できます。

## できること

- 小数のかけ算
- 小数のたし算・ひき算
- 小数のわり算
- 分数のたし算・ひき算
- 分数のかけ算・わり算
- かけ算
- たし算・ひき算
- わり算
- 四捨五入

## 特長

- ブラウザだけで動作します。追加インストールは不要です。
- 問題と模範解答をそれぞれ別に印刷できます。
- A4印刷を前提にレイアウトを調整しています。
- 画面に出る文言は日本語で統一しています。

## 使い方

1. `index.html` をブラウザで開きます。
2. 上部タブから使いたいプリントを選びます。
3. 桁数や問題数などの条件を設定します。
4. 「問題を生成」を押します。
5. 必要に応じて「問題だけ印刷」「模範解答だけ印刷」を使います。

`index.html` は `iframe` で各ページを読み込んでいます。印刷プレビューが合わない場合は、個別のHTMLファイルを直接開いて印刷してください。

## ページ一覧

### index.html

入口ページです。各プリントページをタブで切り替えて表示します。

### syousuu_kakeru.html

小数のかけ算プリントです。

- A・B の整数部と小数部の桁数を指定できます。
- 小数部に `0` を含めるか選べます。
- 10問または20問で出題できます。

### syousuu_tasihiki.html

小数のたし算・ひき算プリントです。

- たし算、ひき算、混合を選べます。
- ひき算は負の答えにならないように調整しています。
- 10問または20問で出題できます。

### syousuu_warizan.html

小数のわり算プリントです。

- A・B の整数部と小数部の桁数を指定できます。
- 解答形式は「小数で答える」「商と余りで答える」「割り切れるまで計算して答える」から選べます。
- 小数で答える場合は、答えの位と丸め方を指定できます。
- 割る数は `0`、`1`、末尾が `0` の小数を自動的に除外します。
- 「割り切れるまで計算して答える」は最大3桁までに制御しています。

### bunsuu_tasihiki.html

分数のたし算・ひき算プリントです。

- 分子と分母の範囲を指定できます。
- 仮分数を許可するか選べます。
- たし算、ひき算、混合を選べます。
- ひき算は負の答えにならないように調整しています。

### bunsuu_kakewari.html

分数のかけ算・わり算プリントです。

- 分子と分母の範囲を指定できます。
- 仮分数を許可するか選べます。
- かけ算、わり算、混合を選べます。
- `1` をかける問題、`1` で割る問題、`0` で割る問題を除外しています。

### seisuu_kakezan.html

整数のかけ算プリントです。

- A・B の桁数を指定できます。
- 10問または20問で出題できます。
- `1` を含む問題を除外する設定があります。

### seisuu_tashihiki.html

整数のたし算・ひき算プリントです。

- たし算、ひき算、混合を選べます。
- 繰り上がり、繰り下がりの有無を設定できます。
- ひき算は負の答えにならないように調整しています。

### warizanmaker.html

整数のわり算プリントです。

- わられる数とわる数の桁数を指定できます。
- 「余りが出てもよい」「割り切れるまで自動調整」を選べます。
- 10問または20問で出題できます。

### shishagonyuu.html

四捨五入プリントです。

- 数の桁や丸める位を指定して出題できます。
- 問題と模範解答を別々に印刷できます。

## ローカル確認

ファイルを直接開いても動作します。必要なら簡易サーバーでも確認できます。

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000/` を開いてください。

## 開発方針

- 画面表示の文言は日本語
- コードの変数名や関数名は英語
- コメントは日本語
- 印刷は `@page { size: A4; margin: 7mm; }` を基準に調整

## 注意点

- ブラウザによって印刷プレビューの見え方が変わることがあります。
- `index.html` 経由の印刷で崩れる場合は、各HTMLを直接開いて印刷してください。
- 画面表示の縦幅が小さい環境では、表示レイアウトが多少変わることがあります。
