# 英検単語テストジェネレーター

## 概要
英検3〜5級を対象とした単語テスト（英→日／日→英）をブラウザだけで生成し、印刷できるシンプルなページです。`index.html` を開くだけで利用でき、乱数シードや出題範囲を調整して同じ問題を再現したり、特定範囲を重点的に学習したりできます。

## 主な機能
- 対象級（3級・4級・5級）の選択
- 10問／20問の出題数切り替え
- 任意文字列・数値から乱数シードを設定して再現性を確保
- 単語番号（No.）の範囲指定によるピンポイント演習
- 英→日／日→英の2種類のテーブルを同時生成し、そのまま印刷可能

## 使い方
1. `index.html` をブラウザで開きます。
2. 対象級・出題数・（必要に応じて）乱数シードと出題範囲を設定します。
   - 出題範囲は該当級の単語リスト番号（1〜`word_count`）で指定します。
3. 「テストを作成する」を押すと 2 種類のテーブルが生成されます。
4. 「印刷」ボタンからPDF出力や紙への印刷が行えます。

## データファイル
各級ごとに `grade*.json` を配置しています。基本構造は以下の通りです。

```json
{
  "grade": "5",
  "source_pdf": "Tangolist/英検５級頻出英単語リスト.pdf",
  "word_count": 100,
  "entries": [
    {
      "number": 1,
      "word": "backpack",
      "pos": "名",
      "meaning": "リュックサック，バックパック"
    }
  ]
}
```

`entries` の `number` を基準に出題範囲を判定します。リストを差し替える際は連番が漏れないようにしてください。

## 開発メモ
- `index.html` の JavaScript で `fetch('grade*.json')` を呼び出しているため、ブラウザによってはローカルファイルを直接開くと読み込み制限にかかる場合があります。その際は簡易的なローカルサーバー（例: `python -m http.server`）を利用してください。
- 乱数シードを未指定の場合はブラウザの `crypto.getRandomValues` を優先使用し、未対応環境では `Math.random` にフォールバックします。
