# 正の数・負の数プリント（加法・減法・乗法・除法）

中学1年生向けの「正の数・負の数」の計算プリントを自動生成する教材ページです。加法・減法・乗法・除法を選んで、問題と模範解答をそれぞれ印刷できます。

## ファイル
- `plusminus.html`: 本体ページ。HTML/CSS/JavaScript を1ファイルにまとめたスタンドアロン構成です。

## 使い方
1. `plusminus.html` をブラウザで開きます。
2. 数の絶対値の範囲、負の数を含めるか、出題する計算の種類を設定します。
3. `10問` または `20問` を選びます。
4. `問題を生成` を押してプリントを作ります。
5. 必要に応じて `問題だけ印刷` または `模範解答だけ印刷` を使います。

## 仕様
- 出題できる計算は `加法` `減法` `乗法` `除法` の4種類です。
- 複数の計算を同時に選ぶと、なるべく均等に出題します。
- 除法は必ず整数で答えが出る問題だけを生成します。
- 除法では割る数に `0` は使いません。
- 負の数は括弧付きで表示し、式が読みやすくなるようにしています。
- 問題と模範解答は画面上で分けて表示し、印刷時はA4縦の2列レイアウトに切り替わります。

## 注意
- 条件が狭すぎると、重複を除いた問題数が足りず、生成に失敗することがあります。
- ブラウザやOSによって印刷時の改ページや余白に微差が出る場合があります。
